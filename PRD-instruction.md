# 문제 정의
생각을 정리하기 위해 마인드맵을 그릴 때, 자료 조사와 생각 정리를 함께 하면 시간이 오래 걸립니다. 또한 주제에 대한 지식 부족으로 누락되는 부분이 생길 수 있습니다. 따라서 사용자가 별도의 지식 없이도 체계화된 마인드맵을 자동으로 작성할 수 있는 서비스가 필요합니다.

# 타켓 사용자 및 사용 사례
시간을 아끼고 생산성을 높이고 싶은 모든 사람들에게 적합.
사례: 기획 단계에서 보조적으로 사용하여 기획 기간을 줄이고 실제 실무 시간에 더 많은 시간과 노력을 할애할 수 있음.

# 이 기능을 사용하는 목표
시간을 아끼고 사전 지식 없이도 모든 분야에서 일정 이상의 품질을 가진 마인드맵을 제작하고 싶을 때.

# 제안 해결책
AI를 사용하여 마인드맵 작성을 자동화하고 사용자는 웹을 통해 언제 어디서든 서비스에 접근할 수 있다.

# 목표
마인드맵의 제작 시간을 낮춰 모두의 생산성을 높인다.
